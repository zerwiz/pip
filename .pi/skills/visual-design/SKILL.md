## Branding and Identity

```bash
# Generate a brand style guide
llm "Create a brand style guide for a sustainable tech company, including color palette (hex), typography, and logo concepts."

# Iconography suggestions
llm "Suggest a set of 10 minimalist icons for a travel booking app."
```

## Layout and Composition

```bash
# Design a landing page hero section
llm "Describe the visual composition for a high-converting landing page hero section for a luxury watch brand."
```

## Workflow

1. **Brand Discovery** — Define the brand values and target audience.
2. **Visual Direction** — Create mood boards or describe the visual style.
3. **Drafting** — Develop initial concepts for logos, colors, and type.
4. **Refinement** — Iteratively improve designs based on feedback.
5. **Asset Export** — Finalize specifications for implementation.

## Notes

- Use consistent color theory and typography.
- Balance whitespace and visual hierarchy.
- Use `vision-language` to analyze existing brand assets.
