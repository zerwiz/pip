## Character Development

```bash
# Develop a character using the Wound/Ghost methodology
llm "Define the 'Wound' (past trauma) and 'Ghost' (haunting memory) for a protagonist in a cyberpunk thriller. How does this drive their internal conflict?"
```

## Story Structure

```bash
# Outline a story using the Hero's Journey
llm "Create a Hero's Journey outline for a story about a young architect discovering a hidden city. Map each of the 12 stages."
```

## Timeline Tracking

```bash
# Track plot consistency across acts
llm "Analyze this plot summary for consistency in character motivations and timeline: [plot content]. Use the Three-Act structure as a baseline."
```

## Workflow

1. **Project Structure Detection** — Identify the current stage of the story (concept, outline, draft).
2. **Character Profile Development** — Use the Wound/Ghost methodology to create deep, motivated characters.
3. **Plot Structuring** — Apply Three-Act or Hero's Journey structures to organize the narrative.
4. **Consistency Checking** — Perform a timeline and motivation audit to ensure narrative integrity.

## Notes

- A character's 'Wound' should be the primary obstacle to their growth.
- Use the 'Ghost' to create recurring themes or symbols in the story.
- Storyboarding is an iterative process; refine as the narrative evolves.
