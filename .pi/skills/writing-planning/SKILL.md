## Structural Outlining

```bash
# Generate a book outline
llm "Create a chapter-by-chapter outline for a non-fiction book about 'The Psychology of Productivity'."

# Character and Setting Development
llm "Create a character profile for the protagonist of a sci-fi novel, including motivations, flaws, and backstory."
```

## Narrative Arcs

```bash
# Plan a story arc
llm "Outline a 'Hero's Journey' narrative arc for a short story about an AI that gains consciousness."
```

## Workflow

1. **Concept Development** — Define the core idea or theme.
2. **Structural Mapping** — Create a high-level outline (chapters, acts, sections).
3. **Detailing** — Flesh out the sub-points or scenes within each structural element.
4. **Refinement** — Ensure the flow is logical and the pacing is appropriate.

## Notes

- Planning saves time during the actual writing phase.
- Use `blog-writing` or `seo-writing` for the implementation phase.
- Useful for technical documentation, novels, and white papers.
