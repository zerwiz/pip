## Slide Deck Outlining

```bash
# Generate a slide deck outline
llm "Create a 10-slide outline for a presentation on 'The Impact of Quantum Computing'. Include slide titles and 3 bullet points per slide."

# Scripting for Slides
llm "Write a 1-minute presenter script for Slide 5 (Future Outlook) of the Quantum Computing presentation."
```

## Visual Composition

```bash
# Suggest visual themes
llm "Suggest a modern, 'tech-forward' color palette and typography for a presentation about AI."
```

## Workflow

1. **Define Objective** — What is the main message of the presentation?
2. **Audience Analysis** — Who are the viewers, and what is their knowledge level?
3. **Outlining** — Structure the flow of information across slides.
4. **Content Drafting** — Write the bullet points and presenter notes.
5. **Design Selection** — Choose a visual theme that complements the content.

## Notes

- Keep slides concise (the '6x6 rule': max 6 bullets, 6 words each).
- Use high-quality visuals to support the text.
- Can be integrated with `chart-creation` for data-driven slides.
