## Scene Planning

```bash
# Generate a storyboard for a 30-second commercial
llm "Create a 6-frame storyboard for a 30-second commercial about a new coffee machine. Include shot descriptions and dialogue."

# Narrative Arc
llm "Outline the narrative arc for a short educational video on 'How Photosynthesis Works'."
```

## Visual Continuity

```bash
# Maintain visual style across scenes
llm "Describe the consistent visual elements (lighting, characters, setting) for a multi-scene animation about space exploration."
```

## Workflow

1. **Script/Concept** — Start with a clear idea or script.
2. **Beat Sheet** — Identify the key moments or 'beats' of the story.
3. **Frame Description** — Detail what happens in each frame (visuals, sound, action).
4. **Iterate** — Refine the sequence for better flow and impact.

## Notes

- Focus on the 'blocking' and composition of each shot.
- Use clear descriptions for camera movements (pan, tilt, zoom).
- Helpful for pre-visualizing complex video projects.
