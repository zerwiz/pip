# Blog Writing

## Setup

```bash
# No special setup. Relies on writing skills and research.
```

## Ideation and Outlining

```bash
# Generate blog post ideas
llm "Suggest 5 catchy blog post titles about 'The Future of Remote Work'."

# Create a detailed outline
llm "Create a detailed outline for a blog post titled 'How to Start a Career in Data Science'. Include sections for skills, tools, and networking."
```

## Drafting and Editing

```bash
# Write a section
llm "Write a compelling introduction for a blog post about 'Top 10 VS Code Extensions'."

# Proofread and Improve
llm "Proofread this draft for grammar, tone, and flow. Suggest improvements to make it more engaging: [draft...]"
```

## Workflow

1. **Ideation** — Brainstorm topics and titles.
2. **Research** — Use `web_search` and `fetch_content` to gather facts and examples.
3. **Outlining** — Structure the post with a logical flow.
4. **Drafting** — Write the content, focusing on voice and value.
5. **Editing** — Refine the text for clarity, SEO, and engagement.

## Notes

- Maintain a consistent brand voice.
- Use `seo-writing` to optimize for search engines.
- Include a clear Call to Action (CTA) at the end.
