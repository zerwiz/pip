## 🚀 How to Use

### Build a Homepage
```bash
pi "Build a homepage for [topic]"
```

### Use Specific Agent
```bash
/design-agent "Create homepage layout"
/content-agent "Write homepage copy"
/dev-agent "Implement homepage code"
/seo-agent "Optimize for search"
```

### Use Complete Team
```bash
pi --team homepage-builder-team "Build complete homepage"
```
